# fantasy-football

An informational fantasy-football **stats & insights** app — you ask questions, an AI **research
analyst** fetches data and answers. Not a league/head-to-head platform.

Built on the **[`agent-harness`](../agent-harness)**: this repo owns the fantasy-football domain
(tools, prompt, data model, UI); the harness owns the model/agent machinery. You bring your own model
(an OpenAI-compatible endpoint).

**Status:** early — scaffolding in progress.
